# waltdisney-mobile-app

## Description
waltdisney-mobile-app is an Android application developed in Kotlin.
The project is based on the Disney universe and provides a mobile interface to explore content in a simple and user-friendly way.

## Features
- Explore Disney-related content
- Mobile application with a simple interface
- Android development with Kotlin
- Structured project for mobile development practice

## Technologies Used
- Kotlin
- Android Studio
- Gradle

## Project Structure
- `app/` : main application source code
- `gradle/` : Gradle configuration
- `build.gradle.kts` : build configuration
- `settings.gradle.kts` : project settings

## How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Let Gradle sync the project
4. Run the application on an emulator or Android device
